# Week 1 – Simple LAN Network Configuration

This repository contains my Week 1 networking assignment, where I created a basic Local Area Network (LAN) using both **Cisco Packet Tracer** and **GNS3**.

## Project Overview

The objective of this task was to design and configure a simple LAN consisting of **four PCs** connected in a **star topology** through a **network switch**.

### Network Configuration

* **Topology:** Star
* **Devices:** 4 PCs and 1 Switch
* **IP Addressing:**

  * PC1 – 195.1.1.1
  * PC2 – 195.1.1.2
  * PC3 – 195.1.1.3
  * PC4 – 195.1.1.4
* **Subnet Mask:** Configured consistently across all devices

After assigning static IP addresses, I performed **ping tests** between the computers to verify successful connectivity across the network. The ping results confirmed that all devices were able to communicate correctly.
